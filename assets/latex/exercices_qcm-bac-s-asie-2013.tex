\begin{h2}Exercice 3   4 points\end{h2}
\textit{Commun  à tous les candidats}
\par
\textit{Les quatre questions de cet exercice sont indépendantes.
     \par
Pour chaque question, une affirmation est proposée. Indiquer si chacune d'elles est vraie ou fausse, en justifiant la réponse. 
\\Une réponse non justifiée ne rapporte aucun point.}
\par
Dans les questions 1. et 2., le plan est rapporté au repère orthonormé direct $\left(O; \vec{u}, \vec{v}\right)$.
\\
On considère les points A, B, C, D et E d'affixes respectives :
\par
$a=2+2i$,     $b =-\sqrt{3}+i$,    $c=1+i\sqrt{3}$,    $d =-1+\frac{\sqrt{3}}{2}i$    et    $e =-1+\left(2+\sqrt{3}\right)i.$
\begin{enumerate}
     \item
     \textbf{Affirmation 1} : les points A, B et C sont alignés.
     \item
     \textbf{Affirmation 2} : les points B, C et D appartiennent à un même cercle de centre E.
     \item
     Dans cette question, l'espace est muni d'un repère $\left(O; \vec{i}, \vec{j}, \vec{k}\right)$.
     \par
     On considère les points $I(1 ; 0 ; 0), J(0 ; 1 ; 0)$ et $K(0 ; 0 ; 1)$.
